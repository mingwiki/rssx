# rssx 命令行查看管理订阅 RSS Feeds

安装

```bash
npm i -g rssx
```

使用

```bash
  rss/rssx {options} {args}
            ls                       列出当前RSS源
            add http://xxx/rss       添加RSS源
            del {index}              删除index对应RSS源
            {index}                  阅读index对应RSS源
                                     打印此帮助信息
```
